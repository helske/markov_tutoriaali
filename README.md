R-koodit artikkelin *Markov-menetelmät elämänkulkuaineistojen tutkimuksessa* analyyseihin
================

- Tiedosto `analyysikoodit.qmd` (Quarto-dokumentti) sisältää empiirisen 
  esimerkin analyysikoodit.
  - Alkuperäinen analysoitu aineisto ei itsessään ole tietoturvasyistä jaettavissa.
  - Tiedosto `simuloidut_sekvenssit.rds` sisältää oikean aineiston ja sen 
  klusteroinnin pohjalta simuloidun synteettisen aineiston jolla analyysikoodeja voi testailla.
  - Synteettisen aineiston simulointiin käytetty koodi löytyy tiedostosta `simuloidun_datan_generointi.R`.
  
- Edellisen analyysin tuottamat kuvat löytyvät kansiosta `kuvat`.

Kaikki mallinnuksessa käytetyt R-paketit on ladattavissa [CRAN](https://cran.r-project.org/)ista.
